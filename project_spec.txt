# Project Specification

## Project Goal
# Path of Exile Knowledge Assistant

A chat application with RAG-powered access to poedb.tw data for Path of Exile 1 and 2.

## Tech Stack

- **Backend:** Python FastAPI with LangChain for RAG pipeline
- **Vector DB:** ChromaDB (local, no external dependencies)
- **Frontend:** React with TypeScript, Tailwind CSS
- **Embeddings:** Local by default (sentence-transformers, Ollama, or LM Studio), OpenAI optional
- **LLM:** Configurable provider support (OpenAI, Anthropic, or local via Ollama)
- **Data Source:** poedb.tw (scraped and indexed on demand)

## Core Concept

```
User Query → API /chat → RAG Query (filtered by game version) → LLM Response with Citations
```

The application maintains a pre-indexed knowledge base from poedb.tw that can be refreshed on demand via an admin endpoint.

## Game Version Filtering

The key feature is filtering RAG queries by game version:

```python
# ChromaDB query with metadata filter
results = vectorstore.similarity_search(
    query=user_query,
    k=3,
    filter={"game": selected_game}  # "poe1" or "poe2"
)
```

Each document in ChromaDB includes:
- `content`: The text content
- `metadata`: {"game": "poe1" | "poe2", "source_url": "...", "title": "..."}
- `embedding`: Vector representation

## Data Flow

1. User selects game version (PoE1 or PoE2)
2. User optionally sets build context (class/ascendancy)
3. User submits query
4. Backend performs RAG search filtered by selected game (and optionally build context)
5. Retrieved context + query sent to LLM
6. LLM generates response with item cards and citations
7. Response streamed back to frontend

## Scraper Implementation

### Target URLs

The scraper will focus on key poedb.tw sections:

- **Items/Uniques:** `/us/Unique/` - All unique items
- **Skills/Gems:** `/us/Gem/` - All skill gems
- **Passive Skills:** `/us/Passive_Skill/` - Tree nodes and passive skills
- **Ascendancy:** `/us/Ascendancy/` - Ascendancy classes
- **Items (Base):** `/us/Item/` - Base item types
- **Modifiers:** `/us/Modifier/` - Map and item modifiers

### Scraper Strategy

1. Start with category index pages
2. Extract links to individual item/skill pages
3. Scrape each page for:
   - Name/title
   - Description/stats
   - Item screenshot/image URL (from poedb.tw or poecdn.com)
   - Base type, requirements, modifiers
   - Flavor text
   - Related items/skills
4. Store in ChromaDB with game metadata and image URLs

### Rate Limiting

- Respect robots.txt and rate limits
- Add delays between requests (1-2 seconds)
- Configurable concurrency

## Design Guidelines

The interface should authentically match Path of Exile's dark, gritty, mature aesthetic:

- **Theme:** Dark mode exclusively - no light mode option
- **Primary Color:** Blood red/crimson (#7b1a1a, #8b0000)
- **Accent Color:** Gold/amber (#c8aa6e, #d4a84b) for highlights, links, and interactive elements
- **Secondary Accent:** Ethereal blue (#4a90d9, #5b9bd5) for magical/skill-related content
- **Background:** Near-black with subtle texture (#0c0c0e to #1a1a1d), subtle gradient overlays
- **Text:** Off-white/cream (#e8e0d5, #d4c4a8) for readability on dark backgrounds
- **Typography:** Clean but slightly stylized - Fontin for headers (or similar fantasy serif), clean sans-serif for body
- **Cards/panels:** Dark with subtle glow borders, gold or red accent borders on hover
- **Borders:** Subtle beveled/gold trim effect on important elements
- **Icons:** Stylized to match PoE UI elements (skill gems, portals, orbs)
- **Visual Effects:** Subtle particle effects, fog/mist in backgrounds, glowing runes
- **Animations:** Smooth, atmospheric - nothing bouncy or playful
- **Overall Feel:** Mature, dark fantasy, Think Wraeclast - grim but functional

### UI Element Styling

- **Buttons:** Dark with gold border, red glow on hover
- **Input fields:** Dark recessed look, gold border on focus
- **Dropdowns:** Dark with gold accent, PoE-style arrow icons
- **Citations:** Stylized like item tooltips with gold borders
- **Loading states:** Subtle glow pulse, not spinners
- **Scrollbars:** Thin, dark with gold accent

```sql
    created_at TIMESTAMP,
    title TEXT
);

CREATE TABLE messages (
    id TEXT PRIMARY KEY,
    conversation_id TEXT,
    role TEXT,  -- 'user' or 'assistant'
    content TEXT,
    citations TEXT,  -- JSON array of sources used
    created_at TIMESTAMP
);
```

## Scraper Implementation

### Target URLs

The scraper will focus on key poedb.tw sections:

- **Items/Uniques:** `/us/Unique/` - All unique items
- **Skills/Gems:** `/us/Gem/` - All skill gems
- **Passive Skills:** `/us/Passive_Skill/` - Tree nodes and passive skills
- **Ascendancy:** `/us/Ascendancy/` - Ascendancy classes
- **Items (Base):** `/us/Item/` - Base item types
- **Modifiers:** `/us/Modifier/` - Map and item modifiers

### Scraper Strategy

1. Start with category index pages
2. Extract links to individual item/skill pages
3. Scrape each page for:
   - Name/title
   - Description/stats
   - Item screenshot/image URL (from poedb.tw or poecdn.com)
   - Base type, requirements, modifiers
   - Flavor text
   - Related items/skills
4. Store in ChromaDB with game metadata and image URLs

### Rate Limiting

- Respect robots.txt and rate limits
- Add delays between requests (1-2 seconds)
- Configurable concurrency

## Design Guidelines

The interface should authentically match Path of Exile's dark, gritty, mature aesthetic:

- **Theme:** Dark mode exclusively - no light mode option
- **Primary Color:** Blood red/crimson (#7b1a1a, #8b0000)
- **Accent Color:** Gold/amber (#c8aa6e, #d4a84b) for highlights, links, and interactive elements
- **Secondary Accent:** Ethereal blue (#4a90d9, #5b9bd5) for magical/skill-related content
- **Background:** Near-black with subtle texture (#0c0c0e to #1a1a1d), subtle gradient overlays
- **Text:** Off-white/cream (#e8e0d5, #d4c4a8) for readability on dark backgrounds
- **Typography:** Clean but slightly stylized - Fontin for headers (or similar fantasy serif), clean sans-serif for body
- **Cards/panels:** Dark with subtle glow borders, gold or red accent borders on hover
- **Borders:** Subtle beveled/gold trim effect on important elements
- **Icons:** Stylized to match PoE UI elements (skill gems, portals, orbs)
- **Visual Effects:** Subtle particle effects, fog/mist in backgrounds, glowing runes
- **Animations:** Smooth, atmospheric - nothing bouncy or playful
- **Overall Feel:** Mature, dark fantasy, Think Wraeclast - grim but functional

### UI Element Styling

- **Buttons:** Dark with gold border, red glow on hover
- **Input fields:** Dark recessed look, gold border on focus
- **Dropdowns:** Dark with gold accent, PoE-style arrow icons
- **Citations:** Stylized like item tooltips with gold borders
- **Loading states:** Subtle glow pulse, not spinners
- **Scrollbars:** Thin, dark with gold accent
